## Available styles

Change the style of `<CTabs>`'s component with modifiers and utilities. Mix and match as needed, or build your own.

### Unstyled

If you don’t provide the `variant` prop, the component will default to a basic style.

<Example code={TabsUnstyledExampleRaw} lang="vue" name="TabsUnstyledExample">
  <TabsUnstyledExample client:only="vue" />
</Example>

### Pills

Take that same code, but use `variant="pills"` instead:

<Example code={TabsPillsExampleRaw} lang="vue" name="TabsPillsExample">
  <TabsPillsExample client:only="vue" />
</Example>

### Underline

Take that same code, but use `variant="underline"` instead:

<Example code={TabsUnderlineExampleRaw} lang="vue" name="TabsUnderlineExample">
  <TabsUnderlineExample client:only="vue" />
</Example>

### Enclosed

Use the `variant="enclosed"` class to give your tab items a subtle border and rounded styling.

<Example code={TabsEnclosedExampleRaw} lang="vue" name="TabsEnclosedExample">
  <TabsEnclosedExample client:only="vue" />
</Example>

### Enclosed pills 

Use the `variant="enclosed-pills"` to achieve a pill-style appearance for each tab item, using pill-shaped borders and smoother outlines.

<Example code={TabsEnclosedPillsExampleRaw} lang="vue" name="TabsEnclosedPillsExample">
  <TabsEnclosedPillsExample client:only="vue" />
</Example>

### Fill and justify

Force your `<CTabs>`'s contents to extend the full available width one of two modifier classes. To proportionately fill all available space use `layout="fill"`. Notice that all horizontal space is occupied, but not every nav item has the same width.

<Example code={TabsUnstyledFillAndJustifyExampleRaw} lang="vue" name="TabsUnstyledFillAndJustifyExample">
  <TabsUnstyledFillAndJustifyExample client:only="vue" />
</Example>

For equal-width elements, use `layout="justified"`. All horizontal space will be occupied by nav links, but unlike the `layout="fill"` above, every nav item will be the same width.

<Example code={TabsUnstyledFillAndJustify2ExampleRaw} lang="vue" name="TabsUnstyledFillAndJustify2Example">
  <TabsUnstyledFillAndJustify2Example client:only="vue" />
</Example>
